La navigation

Par défaut la navigation est horizontale avec un alignement du menu à droite. Pour obtenir ce rendu il suffit d'entourer le menu avec la balise <nav></nav> (Voir le code ci-dessous)

Le menu peut contenir des niveaux imbriqués. Dans ce cas ne pas oublier d'ajouter une icône <i class="icon-angle-down"></i>.

Pour que votre menu soit centré ou à gauche, il suffit d'ajouter --center <nav class="nav--center"></nav> ou --left <nav class="nav--left"></nav>.

Le positionnement et l'alignement du texte des niveaux imbriqués coîncident avec le positionnement de votre navigation.

Exemple : Si votre menu est aligné à droite (par défaut), alors les éléments du sous-menu seront positionnés en bas à droite de l'élement parent.
